# EasyScreenShot (EasyShot)


## 核心优势

- 支持输入截图尺寸
- 支持连续截图
- 支持边调整后方内容边截图

## 安装

```bash
npm install -g .
```

## 使用方法

### 基础截图
```bash
easyshot
```

### 连续截图 (-w)
截图后不退出，保持取景框位置，适合连续截取同一位置的内容
```bash
easyshot -w
```

### 自定义保存路径 (-o)
将截图保存到指定文件夹
```bash
easyshot -o ~/Documents
```

### 查看帮助 (-h)
```bash
easyshot -h
```
